Controls

Click the canvas first, then use the keyboard.

  • 1–8 launch a ball on that lane
  • P toggle orbit / first-person projection
  • W A S D first-person move
  • ↑ ↓ ← → rotate view
  • PgUp / PgDn or wheel — zoom
  • [ / ] camera height
  • L toggle lighting · X toggle axes
  • + / - field of view
  • R roll demo · 0 reset view
  • Left-click drag — rotate
